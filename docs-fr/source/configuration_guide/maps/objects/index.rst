=====================
Les Objets Dynamiques
=====================

Un objet dynamique est un objet dont l'aspect évolue en fonction de l'état de la ressource qu'il représente.

Les ressources disponibles sont les suivantes:

* **Host**
* **Services**
* **Hostgroup**
* **Servicegroup**
* **Carte**
  
3 types d'affichages sont disponibles pour les objets dynamique:

* **Icône**
* **Ligne**
* **Gadget**
